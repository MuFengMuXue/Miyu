# Miyu

Miyu 是一个命令行 AI 助手，面向 Linux 桌面日常使用、系统排障、知识检索和工具自动化。

## 功能

- 终端对话与 REPL，支持 Plan/Yolo 两种工作模式。
- 本地知识库、默认 Linux 桌面知识库和 Shorin ArchLinux Guide 集成。
- AUR/PKGBUILD 审查与安装流程保护。
- 本机问题检查、网页搜索、图片搜索、识图、生图、闹钟、记忆、skills。
- Markdown 终端渲染，支持表格换行、代码块高亮、透明终端友好样式。
- 人格化表情包系统，支持内置表情库、按人格加载、搜索、发送、新增、更新和删除。
